# Security Policy

## Reporting a Vulnerability

Please report vulnerabilities privately via
[GitHub security advisories](https://github.com/Kataglyphis/Kataglyphis-DocumANTation/security/advisories/new)
or by email to <jonasheinle@googlemail.com>. Do not open a public issue for
security reports.

## Supported Versions

Only the latest state of `main` is supported.
